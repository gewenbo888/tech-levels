# Technology Levels L1–L5 — 技术等级

[![Live Demo](https://img.shields.io/badge/Live-tech--levels.vercel.app-8b5cf6)](https://tech-levels.vercel.app)

A bilingual (Chinese–English) maturity framework mapping 11 technological domains from foundational capability (L1) to civilizational transcendence (L5).

## Levels

- **L1 Foundation** — Present-day baseline (Chat, Driver assist, Fossil fuels)
- **L2 Augmentation** — Near-future enhancement (Reasoning, Partial autonomy, Solar)
- **L3 Autonomy** — Conditional self-operation (Agents, DAO, Nuclear fission)
- **L4 Mastery** — High autonomy & innovation (Innovation AI, Fusion, Nanotech)
- **L5 Transcendence** — Civilizational horizon (Organizer AI, Dyson sphere, Warp drive)

## Domains

Foundation Models, Humanoid Robots, Autonomous Driving, Brain–Computer Interface, Web3, Longevity, Energy, Materials, Virtual Reality, Spatial Mobility

## Tech

Single-page HTML/CSS/JS with canvas hero animation, scroll-reveal, and sticky level navigation.
